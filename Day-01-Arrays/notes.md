Day 1 - Array Traversal

Topics Covered:
- Linear Traversal (Left to Right)
- Reverse Traversal (Right to Left)

What I Learned:
- Array traversal means accessing each element in the array one by one.
- In linear traversal, we start from index 0 and go till n-1.
- In reverse traversal, we start from n-1 and go till index 0.
- Traversal can be done using for loop or while loop.
- It is the most basic and important concept in arrays.

Key Concepts:
- Arrays follow 0-based indexing.
- Length of array (n) decides the loop range.
- Forward traversal condition: i < n
- Reverse traversal condition: i >= 0
- Wrong conditions can cause errors like infinite loop or index out of range.

Key Insight:
Traversal is the foundation of Data Structures and Algorithms.
Strong understanding of traversal helps in solving problems easily.

Why This Matters:
- Used in searching algorithms
- Used in sorting algorithms
- Helps in building logic for complex problems
- Important for coding interviews
