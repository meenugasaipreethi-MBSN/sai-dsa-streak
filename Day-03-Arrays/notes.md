# Day 3 - Array Deletion

## Topics Covered
- Deletion at beginning
- Deletion at specific position

## What I Learned
- Deletion in arrays requires shifting elements
- Removing an element affects positions of remaining elements

## Mistake / Realization
Initially, I thought deletion is just removing a value.
But I realized that elements need to be shifted to maintain order.

## Key Insight
Deletion operations help understand how arrays manage memory and positions internally
