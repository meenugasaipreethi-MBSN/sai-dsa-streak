# Day 4 - Find Maximum Element

## Problem
Find the largest element in an array

## Approach
- Assume the first element as maximum
- Traverse the array
- Compare each element with current maximum
- Update maximum when a larger element is found

## What I Learned
- Tracking values during traversal is important
- Even simple problems require careful logic

## Mistake / Realization
Initially, I thought it was very easy,
but handling all cases correctly requires attention

## Key Insight
Strong fundamentals are built by mastering simple patterns

## Time Complexity
O(n)
